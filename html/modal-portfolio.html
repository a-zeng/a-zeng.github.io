Portfolio

A static website page for my portfolio, hosted on GitHub using GitHub Pages

Image placeholder

Objectives:
  • Create a nice-looking, lightweight, and professional static portfolio website
  • Learn more about web design by learning new toolkits and APIs like Bootstrap
Challenges:
  • Aligning containers and divs using HTML and Bootstrap
  • Applying new APIs to the website
  • Determining how to design the website so it does not break when updating projects
Results:
  • Modern static site that loads surprisingly fast despite graphics
  • Skill timeline that is easy to program and update
Takeaways:
  • Website builders aren't necessary to make appealing static sites, such as portfolios
  • Much can be learned from doing it yourself
