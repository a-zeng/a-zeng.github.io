DOSS

A volunteer website redesign for the UMass Lowell's Toxic Use Reduction Institute's (TURI) Database of Safer Solvents (DOSS)

The Database of Safer Solvents (DOSS)

Objectives:
  • Modernize the site
  • Update the database and site formatting to be easily viewable on mobile
  • Adjust page headers and footers to be consistent across all pages
  • Streamline UX
Challenges:
  • Learning PHP and hosting a Apache server for testing
  • Debugging Bootstrap container positioning
  • Incorporating feedback from TURI in a timely manner
Results:
  • DOSS website is easily viewable on mobile without weird sizing
  • Organized file directory that makes links nicer
  • Headers and footers can be easily edited from one file and update to all web pages
  • Database looks less-cluttered and loads faster
  • TURI is satisfied with the work
Takeaways:
  • Learned PHP and how to better use web dev tools in Chrome
  • Real-world server hosting is extremely complex due to compatibility issues and other unexpected pitfalls
  • Web design is fun, although often things don't work as expected
